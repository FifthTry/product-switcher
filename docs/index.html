Product Switcher
-- switcher can be used as product switcher on your fastn web projects.
Package Name:
fifthtry.github.io/product-switcher

Implementation

How to use product-switcher?

To use product-switcher add below line into your FASTN.ftd file:
null
-- fastn.dependency: fifthtry.github.io/product-switcher

-- fastn.dependency: fifthtry.github.io/product-switcher

Invocation:

null
-- import: fifthtry.github.io/product-switcher as ps
